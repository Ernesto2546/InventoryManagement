# 📦 Inventory Management System

A web-based inventory management system built with ASP.NET Core MVC, C#, Entity Framework Core, and SQL Server. The application allows users to manage products, categories, and suppliers through a complete CRUD interface.

## 🚀 Features

- Product management
- Category management
- Supplier management
- Search and filtering
- Dashboard with inventory statistics
- Form validation
- Entity Framework Core Code First
- SQL Server integration

## 🛠️ Technologies

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5
- LINQ
