# 📦 Inventory Management System

A web-based Inventory Management System developed with **ASP.NET Core MVC**, **C#**, **Entity Framework Core**, and **SQL Server**.

This project is being built as part of my software development portfolio to demonstrate backend development, database design, and CRUD operations using the MVC architecture.

## 🚀 Current Features

* ✅ ASP.NET Core MVC project structure
* ✅ Entity Framework Core Code First
* ✅ SQL Server integration
* ✅ Database migrations
* ✅ Category CRUD (Create, Read, Update, Delete)
* ✅ Responsive layout with Bootstrap

## 🛠️ Technologies

* ASP.NET Core MVC
* C#
* .NET
* Entity Framework Core
* SQL Server
* Bootstrap 5

## ▶️ Getting Started

1. Clone the repository.
2. Configure the SQL Server connection string in `appsettings.json`.
3. Run the migrations:

```powershell
Update-Database
```

4. Run the application.
